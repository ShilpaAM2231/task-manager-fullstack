# Task Manager Fullstack Application

A **Full Stack Task Manager** built using **Spring Boot** (Backend) and **React.js** (Frontend).

---

## 🚀 Features

* Create Tasks
* Edit Tasks
* Delete Tasks
* Task Status (Pending / Completed)
* REST API with Spring Boot
* Swagger API Documentation
* React Frontend UI
* Full CRUD Operations

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Axios
* CSS

**Backend**

* Spring Boot
* Spring Data JPA
* H2 Database
* Maven

## ▶ Run the Project

### Backend

```
cd backend
mvn spring-boot:run
```

Runs on:

```
http://localhost:8081
```

---

### Frontend

```
cd frontend
npm install
npm start
```

Runs on:

```
http://localhost:3000
```
